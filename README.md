# CMGUI
一个简单易用的IMGUI库，用于Processing。


A simple easy IMGUI code for [Processing](https://processing.org/) by java language.


GUI elements performed.
* Layout
  - Vertical flow layout
  - Horizontal flow layout
  - Vertical Scroll layout
* Label
* Button
* CheckButton
* Image
* Slider
